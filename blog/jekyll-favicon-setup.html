# 前提

私はGitHub PagesでJekyll/minimaを利用しているので、同様な構成な方に向けた記事です。

# favicon設定

./includes/custom-head.htmlに以下のコードを追記

<link rel="icon" href="/favicon.ico">

用意したfaviconをルートディレクトリに配置。
