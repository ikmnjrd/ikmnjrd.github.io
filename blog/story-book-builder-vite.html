storybook-builder-viteを使ってみる

Posted on 2022-01-09

使ってみたら使用感的にとてもよかった。 ただ、 Getting started with React, Vite and Storybook (on a new project)の中で紹介されていた@vitejs/appがdeprecatedだったので、修正したものを紹介する。

npm init vite@latest vite-react-app -- --template react-ts && cd vite-react-app
npm install # or yarn
npx sb@next init --builder storybook-builder-vite && npm run storybook

関係ないが、viteの公式テンプレート集などもあるのでこれから参考にバイトライフをEnjoyしようと思います
