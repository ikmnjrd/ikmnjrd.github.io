# 結論

.zshenvならvim起動時にも読み込まれるので、エイリアスは.zshenvに記述する。 .config/nvim/init.nvim に以下を記述

set shell=zsh

# 余談

ブログを書くときに"yyyy-mm-dd"という形でいつも日付を書いていた。
その時にvimからdate "+%Y-%m-%d"と実行・挿入したかった。
毎回オプションを書くぐらいならalias(名前day)として登録したが、:r! dayで挿入されなかった. (エラー内容: シェルが値を返しました 127)
